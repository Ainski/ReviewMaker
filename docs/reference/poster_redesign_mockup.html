研究背景与问题定义Background · 摘要节选
随着以 Transformer 架构为代表的大语言模型在自然语言处理领域取得突破性进展，其推理效率与可扩展性正面临严峻挑战。自回归生成过程中，模型需重复计算历史词元的注意力，导致计算冗余与延迟剧增。键值缓存（KV Cache）技术应运而生，将先前词元的 Key/Value 缓存复用，显著减少浮点运算；然而其内存占用随序列长度线性增长，成为 GPU 显存与推理吞吐的关键瓶颈。
